# 'Data Platform'

### Preamble
I received a free trial to the new Codex 5.4 model and decided to put it through its paces (sidebar, first time a promotional email has ever convinced me to sign up to ANYTHING).

I didn't need to build a toy webapp or CRUD application as I know for a fact any half-decent coding model can spit one of those out in an afternoon. I wanted to give it something complex and span a large number of features and see how it performs.

> **Disclaimer** Please be aware, this was in no way scientific; it was for my own enjoyment and self directed learning outcomes. Truthfully, I wanted to see how hands-off I could be and still end up with what I had asked for. <br> I am sure that the prompting could have been higher quality, the work organised piece by piece, carefully divvied up between threads. However, that's not why I was here. I'm on holidays for goodness sake! I wanted to watch some tv and have my laptop on the couch prompting away, so that's exactly what I did.

Despite my lack of vibe coding experience (that feels wrong to say aloud), I do use AI for a number of things; both at work and at home:

- lightning fast, natural language search engine (can be wrong, requires fact checking)
- Generating boilerplate code or snippets in languages I am unfamiliar with
- Onboarding to large codebases and generating *some* documentation (writing is a love of mine and I would be quite disappointed to move this all to llm word-vomit *)
- Recipe ideas when I don't know what to eat for dinner
- Instant crowd source opionins and recomendations (books, tv shows, board games etc)
